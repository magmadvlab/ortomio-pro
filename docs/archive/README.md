# Archivio documentale OrtoMio

**Stato:** historical

L'archivio versionato e intenzionalmente minimo. I piani, report di sessione, messaggi commit, guide deploy e riepiloghi `COMPLETE/SUCCESS` precedenti alla release candidate P0-P9 sono stati assorbiti da `MASTERDOC.md`, dal manuale e dal piano esecutivo corrente.

La ricostruzione puntuale resta disponibile nella cronologia Git precedente al commit di bonifica P9. Un documento storico puo tornare in questa directory solo se conserva una decisione ancora necessaria, indica data e stato `historical` e rimanda alla fonte canonica che lo sostituisce.
