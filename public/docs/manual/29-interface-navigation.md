# Interfaccia e navigazione

[← Indice](./README.md)

La navigazione desktop, mobile, bottom bar, ricerca e Help deriva dallo stesso registro capability. Visibilità significa che ruolo, piano, feature flag, schema e provider richiesti risultano compatibili.

I badge indicano Beta o Simulazione. Admin e configurazione satellite sono riservati agli Admin. Route legacy o tecniche possono reindirizzare all'ingresso canonico e non vanno usate come scorciatoie.

La capability è risolta dal server; un flag locale non autorizza accesso o scrittura.
