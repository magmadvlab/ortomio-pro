# 📖 MANUALE UTENTE ORTOMIO 2026 - INDICE GENERALE

## 🌟 BENVENUTO IN ORTOMIO

OrtoMio è la piattaforma agricola digitale completa che scala automaticamente dal piccolo orto domestico alle grandi aziende agricole. Un'unica soluzione con tecnologie all'avanguardia come AI predittiva, droni, blockchain e certificazioni automatizzate.
