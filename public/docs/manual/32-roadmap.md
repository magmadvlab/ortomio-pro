# Roadmap

[← Indice](./README.md)

La coda corrente è il [piano esecutivo P0-P9](../reports/execution-plans/ORTOMIO_PIANO_ESECUTIVO_COMPLETAMENTO_2026-07-16.md).

La baseline applicativa P0-P8 è completata localmente. Il prossimo backlog operativo è il rollout remoto controllato:

1. snapshot e restore drill sul target autorizzato;
2. applicazione migrazioni e verifica drift;
3. RLS cross-tenant e Security Advisor;
4. smoke dei provider reali;
5. shadow mode e pilot limitato;
6. attivazione progressiva o rollback.

Drone, blockchain, robotica autonoma e certificazione automatica non fanno parte della release candidate.
