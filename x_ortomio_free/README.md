# OrtoMio Free - Componenti Gamification

Questa directory contiene tutti i componenti di gamification, social sharing e ricette che sono stati rimossi da OrtoMio Pro per focalizzarsi sui professionisti.

## 📁 Contenuto

### 🎮 Gamification Components
- `components/garden/ChallengeSection.tsx` - Sezione sfide e gamification
- `components/calendar/IntegratedCalendarWithChallenges.tsx` - Calendario con sfide integrate
- `services/integratedChallengeService.ts` - Servizio per gestione sfide

### 📱 Social Sharing Components
- `components/social/SocialShareModal.tsx` - Modal per condivisione social
- `components/social/ShareButton.tsx` - Pulsante condivisione
- `components/social/SocialStats.tsx` - Statistiche social
- `components/social/PhotoCapture.tsx` - Cattura foto per social
- `services/socialSharingService.ts` - Servizio condivisione social

### 🍳 Recipe Components
- `components/garden/SmartRecipesWidget.tsx` - Widget ricette intelligenti

### 🧪 Testing Components
- `components/testing/SocialSharingTest.tsx` - Test componenti social
- `test-social-sharing.js` - Test script social sharing

## 🎯 Utilizzo in OrtoMio Free

Questi componenti possono essere integrati in OrtoMio Free per offrire:

1. **Gamification**: Sfide, punti XP, badge per coinvolgere utenti hobbisti
2. **Social Sharing**: Condivisione raccolti e progressi sui social media
3. **Ricette**: Suggerimenti culinari basati sui raccolti
4. **Community**: Elementi per creare una community di appassionati

## 🔧 Integrazione

Per integrare questi componenti in OrtoMio Free:

1. Copiare i file nella struttura appropriata
2. Installare le dipendenze necessarie
3. Aggiornare i percorsi di import
4. Configurare i servizi social (API keys, etc.)
