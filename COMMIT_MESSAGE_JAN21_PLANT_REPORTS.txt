## New Features

### 📊 Plant Reports Mock Page
- Created `/app/reports` page with realistic mock data
- Shows complete crop cycle history with real-world examples
- Interactive crop selector (Tomato San Marzano / Romaine Lettuce)
- Three view modes: Summary, Detailed, Comparison

### 📋 Summary View Components
- **Crop Information Card**: Name, variety, location, planting/harvest dates, cycle duration
- **4 KPI Cards**: Yield (18.5kg), Quality (4.5/5), Revenue (€240), ROI (+182%)
- **Operations Timeline**: 7 operations with icons and color coding
  - Soil preparation (tilling)
  - Transplanting
  - Fertilization
  - Irrigation
  - Problem detection (aphids)
  - Treatment
  - Harvest
- **Cost Analysis**: Breakdown with progress bars (preparation, plants, fertilizers, treatments, irrigation)
- **Economic Summary**: Revenue, costs, profit, ROI calculation
- **Problems & Solutions**: Issue tracking with resolution status
- **Additional Stats**: Photos, operations count, weather averages

### 🎨 Design Features
- Color-coded operation types (blue, green, purple, red, orange, yellow)
- Gradient KPI cards with icons
- Responsive mobile-friendly layout
- Professional UI with Tailwind CSS
- Export PDF button (UI only)
