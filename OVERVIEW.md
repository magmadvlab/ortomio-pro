# 🌱 OrtoMio AI - Panoramica

## Cos'è OrtoMio?

**OrtoMio** è un'applicazione web intelligente progettata per aiutare gli appassionati di orticoltura a gestire il proprio orto in modo efficiente e sostenibile. Combinando l'intelligenza artificiale (Google Gemini), dati meteorologici in tempo reale, e conoscenze agronomiche tradizionali, OrtoMio offre un assistente virtuale completo per la gestione dell'orto domestico.

## Il Problema che Risolve

Gestire un orto richiede conoscenze specifiche su:
- **Quando** seminare e trapiantare
- **Come** curare le piante in ogni fase di crescita
- **Cosa** fare in caso di problemi o malattie
- **Quanto** irrigare e concimare
- **Dove** posizionare le piante per ottimizzare lo spazio

OrtoMio automatizza e semplifica tutte queste decisioni, fornendo consigli personalizzati basati sulla posizione geografica, il tipo di terreno, e le condizioni meteorologiche.

## Funzionalità Principali

### 🏠 Dashboard (Home)
- **Panoramica generale** dell'orto con statistiche in tempo reale
- **Widget meteo** con previsioni e allarmi (gelate, caldo estremo, piogge intense)
- **Lavori preparatori invernali** con piano strutturato per stagione
- **Fabbisogno idrico giornaliero** calcolato per tutto l'orto
- **Photo reminders** automatici ogni 15 giorni per tracciare i progressi
- **Modalità vacanza** con piano di sopravvivenza per le piante

### 🌱 Planner (Semina)
- **Suggerimenti stagionali** basati su posizione GPS e clima locale
- **Ricerca piante specifiche** con informazioni dettagliate
- **Guide "For Dummies"** passo-passo per semina, trapianto e raccolta
- **Distinzione Dal Seme / Dalla Piantina** con guide specifiche
- **Consociazioni consigliate** (piante che si aiutano a vicenda)
- **Visual Garden Planner** per progettare graficamente l'orto
- **Calendario lunare** integrato per semine tradizionali
- **Banca Semi** con gestione inventario e scadenze

### 📔 Journal (Diario)
- **Timeline attività** con foto e note
- **Lifecycle Coach** che monitora le fasi di crescita (germinazione, trapianto, produzione)
- **Nutrient Coach** che calcola il fabbisogno NPK dinamico
- **Health Coach** con suggerimenti preventivi per trattamenti
- **Analisi AI delle foto** per diagnosi visiva di problemi
- **Promemoria automatici** per attività ricorrenti

### 🛡️ Advice (Cura)
- **Diagnosi problemi** tramite descrizione testuale o foto
- **Suggerimenti trattamenti** bio e tradizionali
- **Auto-scheduling** dei trattamenti preventivi
- **Calendario trattamenti** con date automatiche
- **Prodotti fitosanitari** con dosaggi e frequenze

### 🧺 Harvest (Raccolto)
- **Registrazione raccolti** parziali e finali
- **Analisi economica** (valore generato, costi, risparmio netto)
- **Ricette AI** tradizionali italiane basate sul raccolto
- **Statistiche per coltura** con resa e qualità
- **Storico completo** di tutti i raccolti

### 📡 Smart Hub
- **Simulazione sensori IoT** per umidità terreno
- **Irrigazione intelligente** con consigli AI
- **Monitoraggio real-time** delle condizioni
- **Controllo valvole** automatico/manuale

## Tecnologie Utilizzate

- **Frontend**: React 19, TypeScript, Tailwind CSS
- **AI**: Google Gemini 2.5 Flash per suggerimenti e analisi
- **Meteo**: Open-Meteo API per previsioni e allarmi
- **Geolocalizzazione**: Browser Geolocation API
- **Storage**: LocalStorage per persistenza dati
- **Build**: Vite per sviluppo e produzione

## Requisiti

- **Browser moderno** (Chrome, Firefox, Safari, Edge)
- **Connessione internet** per AI e meteo
- **Permessi geolocalizzazione** (opzionale, migliora i suggerimenti)
- **API Key Gemini** (opzionale, per funzionalità AI avanzate)

## Setup Rapido

1. Clona il repository
2. Installa dipendenze: `npm install`
3. Configura API Key (opzionale): crea `.env` con `VITE_GEMINI_API_KEY=your_key`
4. Avvia: `npm run dev`
