## 🧮 ALGORITMI IMPLEMENTATI

### **Optimization Algorithms**

#### **1. Genetic Algorithm**
```typescript
// Multi-objective fitness evaluation
const fitness = (
  objectives.minimizeCost * (1 / cost) +
  objectives.maximizeYield * yield +
  objectives.minimizeEnvironmentalImpact * environmental +
  objectives.maximizeEfficiency * efficiency
);
```

#### **2. Simulated Annealing**
```typescript
// Temperature-based acceptance
const accept = deltaFitness > 0 || 
  Math.random() < Math.exp(deltaFitness / temperature);
```

#### **3. Particle Swarm Optimization**
```typescript
// Velocity and position updates
velocity = w * velocity + c1 * r1 * (pBest - position) + 
           c2 * r2 * (gBest - position);
```

#### **4. Gradient Descent**
```typescript
// Gradient-based optimization
position = position - learningRate * gradient;
```
