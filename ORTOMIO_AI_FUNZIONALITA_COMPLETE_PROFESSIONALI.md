## 🌞 Sistema Solare e Microclimi Avanzati

### Calcoli Astronomici Precisi
Il sistema implementa formule astronomiche complete per calcoli solari:

#### **Posizione Solare Real-Time**
- **Declinazione solare**: `23.45° × sin(360° × (284 + dayOfYear) / 365)`
- **Equazione del tempo**: correzione longitudine e fuso orario
- **Elevazione solare**: calcolo altezza sopra orizzonte (0-90°)
- **Azimut solare**: direzione (0-360°, 0=Nord, 90=Est, 180=Sud, 270=Ovest)
- **Step temporali**: calcoli ogni 10 minuti per precisione massima

#### **Gestione Ostacoli 3D**
```typescript
interface Obstacle3D {
  azimuth: number;        // Direzione ostacolo (0-360°)
  height: number;         // Altezza in metri
  distance: number;       // Distanza orizzontale
  widthDegrees: number;   // Larghezza angolare
  type: 'Building' | 'Tree' | 'Mountain' | 'Other';
}

// Calcolo blocco solare
const obstacleElevation = Math.atan2(height, distance) * 180 / Math.PI;
const isBlocked = sunElevation < obstacleElevation;
```

#### **Estrazione Automatica da Foto 360°**
- **Analisi AI** per identificazione automatica ostacoli
- **Correzione offset Nord** per orientamento preciso
- **Classificazione automatica**: edifici, alberi, montagne
- **Stima intelligente** altezza/distanza basata su tipo
- **Supporto sorgenti multiple**: foto_360, manual, ai_analysis

### Classificazione Orto Intelligente
Il sistema classifica automaticamente ogni area in:

#### **Orto Estivo** (>6h sole Giu-Lug)
- **Colture ottimali**: pomodori, peperoni, zucchine, melanzane, cetrioli
- **Sistemi consigliati**: irrigazione a goccia, pacciamatura
- **Periodo produttivo**: Aprile-Ottobre

#### **Orto Non Estivo** (<6h sole Giu-Lug)  
- **Colture ottimali**: lattughe, spinaci, rucola, bietole, cavoli
- **Sistemi consigliati**: irrigazione micro-spray, protezioni
- **Periodo produttivo**: Settembre-Maggio

#### **Orto Misto** (5-7h sole variabile)
- **Colture ottimali**: fagiolini, piselli, carote, cipolle, erbe
- **Sistemi consigliati**: irrigazione mista, rotazioni stagionali
- **Periodo produttivo**: tutto l'anno con rotazioni
